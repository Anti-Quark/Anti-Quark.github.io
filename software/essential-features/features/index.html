Essential Features

for Minecraft Forge

Features:

  • The viewed block, which emits redstone when looked at.
  • The mechanic villager.
  • Stained Redstone lamps.
  • Snow Brick.
  • Smooth Glowstone.
  • Polished Glowstone.
  • Cereal.
  • Iron-Fortified Cereal.
  • Decorative Stone, Granite, Diorite, and Andesite.
  • Block Breaker: Provide Redstone and it will break the block in front of it!
  • Crying Obsidian. It functions like a bed, without having to sleep on it - as soon as you place it, your respawn place is set to it.
  • Particles and sounds on respawn.
  • Spike Block. You take damage from it, and also slows you down. You can walk through it.
  • Seamless Double Slabs (Stone, Sandstone, and Red Sandstone).
  • Block Placer: Provide Redstone and it places blocks from its inventory.
  • More Bricks (Cream Bricks, London Bricks, Roman Bricks, Blue Bricks, and Mixed Bricks) as well as items used to craft them.
