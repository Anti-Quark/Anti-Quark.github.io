Essential Features mod for Minecraft 1.10

Essential Features mod adds things which we believe are essential additions to minecraft. It adds features which a) are vanilla-like, and b) probably won't be added in the near future.

It currently adds two blocks, the Viewed Block, and Smooth Glowstone. When looked at, the Viewed Block will emit a redstone signal. Smooth Glowstone is the same as glowstone, but with a smooth texture.
